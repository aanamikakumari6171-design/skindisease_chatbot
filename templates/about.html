About SkinCheck AI

SkinCheck AI is an AI-powered web application designed to help users detect possible skin diseases within seconds.

Features:

  • Upload image or describe symptoms
  • Instant detection using AI model
  • Voice input for ease of use
  • Simple and user-friendly interface

Note: This tool is for educational and awareness purposes only. It is not a replacement for certified medical diagnosis.
